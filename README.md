# WatchMe

멀리 떨어져 있는 친구나 연인이 같은 유튜브 영상을 함께 보면서, 같은 화면 안에서 채팅과 반응을 주고받는 웹 서비스 MVP입니다.

## 현재 구현 상태

- 유튜브 링크로 같이 보기 방 만들기
- 초대 링크로 방 입장
- 닉네임 기반 참여
- 유튜브 플레이어 표시
- 실시간 채팅과 빠른 이모지 반응
- 참여자 목록과 상태 표시
- 방장 중심 재생 권한과 권한 위임
- 광고/로딩/네트워크 지연으로 시간이 어긋날 때 `잠깐 쉬는 타임, 이어보시겠어요?` 팝업 표시
- `기다릴게요`, `이어보기`, `현재 방 시간으로 맞추기` 흐름

## 실행 방법

```bash
npm install
npm run dev
```

실행 후 브라우저에서 아래 주소로 접속합니다.

```text
http://localhost:3000
```

## 참고

첫 버전은 유튜브 광고를 제거하거나 직접 제어하지 않습니다. 유튜브 플레이어 안에서 광고가 재생되는지는 사용자별 계정, 프리미엄 여부, 브라우저 상황에 따라 달라질 수 있으므로, 현재 MVP는 시간 차이를 감지해 사용자에게 선택지를 보여주는 방식으로 대응합니다.

## Vercel 모바일 프론트 확인

Vercel 배포는 모바일 화면 확인을 위한 정적 프론트 미리보기 용도입니다. Vercel에서는 현재 Express/Socket.IO 실시간 서버를 그대로 유지하지 않기 때문에, 배포된 화면에서는 방 생성 시 `프론트 미리보기` 방으로 이동합니다.

로컬 개발 서버에서는 기존처럼 실시간 방 기능을 확인할 수 있습니다.
