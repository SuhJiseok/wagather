# WatchMe 기획 문서

## 제품 목표

WatchMe는 멀리 떨어진 사람들이 같은 유튜브 영상을 거의 같은 시점으로 보면서, 별도 메신저로 이동하지 않고 바로 채팅과 반응을 남길 수 있게 하는 같이 보기 서비스입니다.

## 핵심 사용자

- 장거리 연인
- 멀리 사는 친구들
- 온라인으로 같이 콘텐츠를 보고 싶은 소규모 그룹

## MVP 범위

현재 버전에서 포함하는 기능은 아래와 같습니다.

- 유튜브 링크로 방 만들기
- 초대 링크 공유
- 닉네임으로 입장
- 같은 영상 시청
- 채팅과 빠른 이모지 반응
- 참여자 상태 표시
- 방장 중심 재생 권한
- 방장이 재생을 누르면 3초 카운트다운 후 같이 재생
- 일반 참여자의 로컬 재생, 정지, 구간 이동 허용

현재 버전에서 제외하는 기능은 아래와 같습니다.

- 회원가입
- 친구 목록
- 음성 통화
- 화상 통화
- 영상 직접 업로드
- 유튜브 외 플랫폼 지원
- 채팅 기록 영구 저장
- 광고 제거 또는 광고 건너뛰기

## 광고/지연 대응 정책

유튜브 광고는 사용자마다 다르게 나올 수 있습니다. 예를 들어 한 사람은 유튜브 프리미엄이라 광고 없이 보고, 다른 사람은 중간 광고를 볼 수 있습니다.

WatchMe는 광고를 직접 감지하거나 제어하지 않습니다. 일반 참여자가 광고나 로딩 때문에 영상을 멈추거나 구간을 이동해도 해당 조작은 본인 화면에만 적용되고, 방장이나 다른 참여자의 영상에는 영향을 주지 않습니다. 이때 본인 화면은 `개인 조작 중`으로 표시되며, `현재 방 시간으로 맞추기`를 눌러 방장 기준 시간으로 돌아올 수 있습니다.

## 재생 권한 정책

방을 개설한 사람이 방장이 됩니다. 방장만 방 전체의 재생, 정지, 구간 이동 기준을 바꿀 수 있습니다.

방장이 처음 재생을 누르면 바로 영상이 시작되지 않고, 모든 참여자에게 3초 카운트다운 팝업을 보여준 뒤 방 기준 시간에서 함께 재생합니다.

방장이 정지, 재생, 구간 이동을 하면 참여 중인 다른 사람의 영상도 같은 상태와 시간으로 맞춰집니다. 일반 참여자가 유튜브 플레이어에서 로컬로 재생, 정지, 구간 이동을 해도 방 전체에는 반영되지 않고 본인 화면에만 적용됩니다.

## 성공 기준

- 사용자가 유튜브 링크 하나로 방을 만들 수 있습니다.
- 초대받은 사람이 링크와 닉네임만으로 입장할 수 있습니다.
- 영상 옆 또는 아래에서 채팅과 이모지 반응을 할 수 있습니다.
- 방장이 재생을 누르면 3초 카운트다운 후 같이 시작됩니다.
- 방장의 정지, 재생, 구간 이동은 다른 참여자에게 동시에 반영됩니다.
- 일반 참여자의 로컬 조작은 방 전체 재생 상태를 바꾸지 않습니다.
- 일반 참여자는 본인 화면에서만 재생, 정지, 구간 이동을 할 수 있습니다.
- 개인 조작 중인 일반 참여자는 방장 명령을 받으면 자동으로 방장 기준 시간으로 돌아옵니다.
