# WatchMe 기획 문서

## 제품 목표

WatchMe는 멀리 떨어진 사람들이 같은 유튜브 영상을 거의 같은 시점으로 보면서, 별도 메신저로 이동하지 않고 바로 채팅과 반응을 남길 수 있게 하는 같이 보기 서비스입니다.

## 핵심 사용자

- 장거리 연인
- 멀리 사는 친구들
- 온라인으로 같이 콘텐츠를 보고 싶은 소규모 그룹

## MVP 범위

현재 버전에서 포함하는 기능은 아래와 같습니다.

- 유튜브 링크로 방 만들기
- 초대 링크 공유
- 닉네임으로 입장
- 같은 영상 시청
- 채팅과 빠른 이모지 반응
- 참여자 상태 표시
- 방장 중심 재생 권한
- 방장이 다른 사람에게 조작 권한 부여
- 광고/지연으로 인한 시간 차이 안내

현재 버전에서 제외하는 기능은 아래와 같습니다.

- 회원가입
- 친구 목록
- 음성 통화
- 화상 통화
- 영상 직접 업로드
- 유튜브 외 플랫폼 지원
- 채팅 기록 영구 저장
- 광고 제거 또는 광고 건너뛰기

## 광고/지연 대응 정책

유튜브 광고는 사용자마다 다르게 나올 수 있습니다. 예를 들어 한 사람은 유튜브 프리미엄이라 광고 없이 보고, 다른 사람은 중간 광고를 볼 수 있습니다.

WatchMe는 광고를 직접 감지하거나 제어하지 않습니다. 대신 상대방 영상 시간이 멈추거나 느려져서 내 영상이 앞서가면, 앞서 있는 사람에게 아래 팝업을 보여줍니다.

```text
잠깐 쉬는 타임, 이어보시겠어요?
```

버튼은 아래 2개입니다.

- `기다릴게요`: 상대방이 따라올 때까지 멈춰 있습니다.
- `이어보기`: 본인만 계속 먼저 봅니다.

`이어보기`를 누른 사용자는 `먼저 보는 중` 상태가 됩니다. 이때 그 사람의 재생 조작은 방 전체 기준 시간을 흔들지 않습니다. 다시 같이 보고 싶으면 `현재 방 시간으로 맞추기`를 누릅니다.

## 성공 기준

- 사용자가 유튜브 링크 하나로 방을 만들 수 있습니다.
- 초대받은 사람이 링크와 닉네임만으로 입장할 수 있습니다.
- 영상 옆 또는 아래에서 채팅과 이모지 반응을 할 수 있습니다.
- 시간이 어긋났을 때 누가 앞서고 뒤처졌는지 알 수 있습니다.
- 광고나 로딩 상황에서도 사용자가 기다릴지 먼저 볼지 선택할 수 있습니다.

## 다음 개선 후보

- 광고/지연 팝업이 너무 자주 뜨지 않도록 빈도 제한
- 채팅 메시지를 영상 시간대별로 다시 볼 수 있는 기능
- 방 삭제와 만료 시간 안내
